# Public Assets

## NFT Image

**Required File:** `nft-preview.png`

This should be the "HAUS OF 1000 - LIMITED EDITION" NFT image with:
- Digital tunnel/vortex background (blue/black)
- Floating ETH teardrop icons
- "HAUS OF 1000" text in center
- "- LIMITED EDITION -" subtitle

**Specifications:**
- Format: PNG (supports transparency)
- Recommended size: 800x800px or 1000x1000px
- Aspect ratio: 1:1 (square)
- File size: < 500KB for best performance

**To add your image:**
1. Save your NFT design as `nft-preview.png`
2. Place it in this `public/` folder
3. Refresh the website

The image will automatically display in the "Donate" section with a glowing blue border.

**Fallback:**
If the image is not found, a placeholder with "HAUS OF 1000 - LIMITED EDITION" text will show instead.
